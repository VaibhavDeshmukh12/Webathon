Responsive App Designing Cost: 449Rs

This course will teach you how to design responsive apps that look great on any device. You will learn the basics of responsive design, including how to use flexbox and grid layouts, how to create responsive images and videos, and how to test your app for responsiveness. You will also learn how to use design tools such as Figma and Sketch to create your app designs.

Course Objectives:

  • Understand the principles of responsive design.
  • Create responsive images and videos.
  • Use flexbox and grid layouts to create responsive layouts.
  • Test your app for responsiveness.

Prerequisites:

  • Basic knowledge of HTML and CSS
  • A design tool such as Figma or Sketch

Certificate:

  • Upon successful completion of this course, you will be awarded a certificate of completion. This certificate can be used to demonstrate your skills to potential employers or clients.
